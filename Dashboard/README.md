# Customer Churn Dashboard
 
A Streamlit dashboard for analysing customer churn patterns, risk segmentation, and ML model performance.
 
## Requirements
 
**Python version:** 3.8 or higher
 
**Required libraries:**
 
```
streamlit
pandas
numpy
plotly
```
 
Install the missing library:
 
```bash
pip install streamlit 
pip install pandas
pip install numpy
pip install plotly
```
 
## Files needed
 
Place all of the following in the same folder before running:
 
```
app.py
churn_data_processed.csv
model_results.csv
```
 
 
## How to run
 
```bash
streamlit run app.py
```
 
The app will open in your browser at `http://localhost:8501`.
